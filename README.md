# Prettify Package for Atom

Prettify your HTML

![Prettify](https://f.cloud.github.com/assets/1238468/2282365/dd9ad7c4-9fac-11e3-90a0-a6778ed81cfe.gif)


### Installing

##### Command Line

    apm install atom-prettify
    
##### Atom

  1. Open settings with `Cmd+,`
  2. Go to `Packages`
  3. Search for `atom-prettify`
  4. Click Install
